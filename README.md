# 🎬 WA Status Video Cutter V2

Ein praktisches Tool für Android, um lange Videos perfekt für den WhatsApp-Status vorzubereiten. 

## ✨ Features
* **61-Sekunden-Schnitt:** Schneidet Videos automatisch in Segmente, die perfekt in den WhatsApp-Status passen (inklusive 1 Sekunde Überlappung für flüssige Übergänge).
* **Einfache Bedienung:** Video auswählen, Start drücken, fertig!
* **Spenden-Dialog:** Unterstütze die Weiterentwicklung direkt per Bitcoin (QR-Code integriert).
